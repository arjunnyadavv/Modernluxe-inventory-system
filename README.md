# ModernLuxe Inventory System

A simple and efficient inventory management system for a clothing store, built using HTML, CSS, JavaScript, and Node.js.

## Features
- Add products
- View stock
- Simple REST API
- Real-world retail use case

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## How to Run
1. Install Node.js
2. Run `npm install`
3. Run `node backend/server.js`
4. Open frontend/index.html
